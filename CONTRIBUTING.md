# Contributing

Thanks for your interest in contributing.

## How to contribute

1. Open an issue describing the problem or proposal.
2. Fork the repository and create a focused branch.
3. Keep pull requests small and explain user impact.
4. Include updates to docs/examples when behavior changes.

## Development

```bash
npm install
npm run prioritize
```

For `timebank-protocol`, run:

```bash
npm run balances
```

## Pull request checklist

- [ ] Problem is clearly described.
- [ ] Change is covered by runnable example/script.
- [ ] README/docs updated when needed.
- [ ] Backward compatibility considered.

## Community expectations

Please read `CODE_OF_CONDUCT.md` before participating.
