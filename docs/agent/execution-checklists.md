# Execution Checklists

## Implementation

- [ ] Story ID selected.
- [ ] Scope limited to one protocol objective.
- [ ] Ledger invariants reviewed.
- [ ] Tests and docs in same PR.

## Validation

- [ ] Run backend tests.
- [ ] Run frontend build/tests.
- [ ] Run ledger reproducibility checks.
- [ ] Validate contracts/examples updated.

## PR

- [ ] Include fairness/trust impact statement.
- [ ] Include risk and rollback notes.
- [ ] Include command evidence.
- [ ] Link issue/story.

## Release

- [ ] Changelog updated.
- [ ] Policy assumptions documented.
- [ ] Known failure modes updated.
- [ ] Pilot playbook aligned.
