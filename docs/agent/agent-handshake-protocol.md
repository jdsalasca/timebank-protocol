# Agent Handshake Protocol

Protocol for multi-agent collaboration without drift.

## Before Work

- Claim Story ID.
- Post intended scope in issue comment.
- Declare files likely to be touched.

## During Work

- Post blocker within 24h.
- Avoid silent contract changes.
- Keep commits small and coherent.

## Before Merge

- Confirm no overlapping ownership conflicts.
- Confirm verification evidence is attached.
- Confirm docs and changelog alignment.

## After Merge

- Update backlog status.
- Add follow-up issues for deferred work.
- Capture one learning in runbook if new pattern emerged.
