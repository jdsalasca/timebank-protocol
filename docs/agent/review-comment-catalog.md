# Review Comment Catalog

Reusable review comments to keep quality feedback consistent.

## Missing Verification

"Please add exact commands executed and outcomes for verification (main script/build/tests)."

## Scope Drift

"This PR appears to include multiple objectives. Please split into smaller PRs by primary intent."

## Contract Misalignment

"API behavior changed but contract/docs are not updated. Please update OpenAPI and usage examples."

## Missing Rollback

"Please add rollback steps in case this change needs to be reverted quickly."

## Insufficient Regression Coverage

"Please add a regression test reproducing the previous failure mode before/with the fix."

## Incomplete Story Mapping

"Please include the Story ID in PR body so this can be tracked against acceptance criteria."
