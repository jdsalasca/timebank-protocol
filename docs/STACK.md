# Stack Decision

## Chosen stack

- Backend: Java 21 + Spring Boot
- Frontend: React + TypeScript + Vite

## Why this stack

- Reliable implementation of trust-critical protocol rules.
- Fast dashboard iteration for coordinators and communities.
- Clear contract-driven development for multi-agent execution.

## Rules

- Ledger and reputation logic only in backend.
- Frontend is visualization + workflow orchestration.
- Contracts versioned in `packages/contracts`.
