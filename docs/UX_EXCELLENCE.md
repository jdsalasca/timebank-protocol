# UX Excellence

## Core principles

- Clarity first, decoration second.
- Every table/list must explain why values matter.
- Show loading, empty, error, and stale states.
- Optimize for mobile and low-bandwidth usage.

## Information architecture

- Surface top priorities at first glance.
- Group data by actionability.
- Keep critical actions above the fold.

## Accessibility baseline

- Good color contrast.
- Keyboard reachable controls.
- Semantic HTML structure.
- Readable text size on mobile.

## Trust UX

- Show update timestamps.
- Explain scoring or alert logic in plain language.
- Never hide uncertainty indicators.
