# Monorepo Guide

## Structure

- `apps/api-java` backend
- `apps/web-react` frontend
- `packages/contracts` shared contracts
- `infra` local runtime
- `docs` strategy and governance

## Agent workflow

1. Pick `agent-ready` issue.
2. Implement backend rules first.
3. Update contract definitions.
4. Implement frontend behavior.
5. Update examples and docs.
